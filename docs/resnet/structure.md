## 关键信息汇总

| 项目 | 数值 |
|------|------|
| **模型** | ResNet18 |
| **输入尺寸** | `(1, 3, 224, 224)` |
| **输出类别** | 1000 (ImageNet) |
| **BasicBlock 数** | 8 个 (Layer1×2 + Layer2×2 + Layer3×2 + Layer4×2) |
| **下采样位置** | Stem (×4), Layer2首块 (×2), Layer3首块 (×2), Layer4首块 (×2) |
| **总下采样倍数** | 32× (224 → 7) |
| **融合说明** | 所有 `Conv2d` 均已融合 `BatchNorm`，`@weight` / `@bias` 是融合后的权重/偏置 |
| **残差连接** | `pnnx.Expression` 算子执行 `add(@0,@1)`，对应 ResNet 的 `F.relu(x + shortcut)` |
