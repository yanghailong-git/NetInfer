## е…ій”®жЁЎеқ—иҜҙжҳҺ

### 1. Inverted Residual BlockпјҲеҖ’ж®Ӣе·®жЁЎеқ—пјү

MobileNetV2 зҡ„ж ёеҝғеҚ•е…ғпјҢе’Ң ResNet зҡ„ж®Ӣе·®еқ—**зӣёеҸҚ**пјҡ

| еҜ№жҜ” | ResNet BasicBlock | MobileNetV2 InvertedResidual |
|------|------------------|------------------------------|
| з»“жһ„ | е®Ҫ вҶ’ **зӘ„** вҶ’ е®Ҫ | **зӘ„** вҶ’ е®Ҫ вҶ’ зӘ„ |
| еҚ·з§Ҝ | 3Г—3 вҶ’ 3Г—3 | 1Г—1 expand вҶ’ 3Г—3 dw вҶ’ 1Г—1 project |
| жҝҖжҙ» | ReLU е…ЁзЁӢ | ReLU6 д»…еңЁ expand/dwпјҢproject еҗҺж— жҝҖжҙ» |
| ж®Ӣе·® | е§Ӣз»Ҳжңү | д»…еҪ“ `s=1` дё” `in_ch == out_ch` ж—¶жүҚжңү |

> `groups=in_channels` зҡ„ `Conv2d` е°ұжҳҜ **Depthwise Separable Convolution** зҡ„ depthwise йғЁеҲҶпјҢи®Ўз®—йҮҸиҝңе°ҸдәҺж ҮеҮҶеҚ·з§ҜгҖӮ

### 2. еҸӮж•° `t` зҡ„еҗ«д№ү

`t`пјҲexpansion factorпјүжҳҜжү©еұ•зі»ж•°пјҡ
- йҰ–еұӮ 1Г—1 е°ҶйҖҡйҒ“д»Һ `c_in` жү©еұ•еҲ° `t Г— c_in`
- 3Г—3 depthwise еңЁжү©еұ•еҗҺзҡ„й«ҳз»ҙз©әй—ҙжҸҗеҸ–зү№еҫҒ
- жң«еұӮ 1Г—1 е°ҶйҖҡйҒ“еҺӢзј©еӣһ `c_out`

дҫӢеҰӮ features.2пјҡ`t=6`, `c=24`пјҢиҫ“е…Ҙ 16 вҶ’ expand еҲ° 96 вҶ’ depthwise вҶ’ project еӣһ 24гҖӮ

### 3. зәҝжҖ§з“¶йўҲпјҲLinear Bottleneckпјү

features.17 зҡ„ project еҚ·з§ҜеҗҺ**жІЎжңү ReLU6**пјҡ
```cpp
nn.Conv2d convbn2d_50 960вҶ’320, 1Г—1
// жіЁж„ҸпјҡиҝҷйҮҢзӣҙжҺҘиҝһеҲ°дёӢдёҖдёӘеҚ·з§ҜпјҢжІЎжңү ReLU6
```

еҺҹеӣ пјҡReLU дјҡжҲӘж–ӯиҙҹж•°пјҢеңЁдҪҺз»ҙз©әй—ҙпјҲ320 йҖҡйҒ“пјүйҖ жҲҗдёҚеҸҜйҖҶзҡ„дҝЎжҒҜжҚҹеӨұгҖӮи®әж–ҮдёӯжҸҗеҮәеңЁ project еұӮдҝқз•ҷзәҝжҖ§жҝҖжҙ»гҖӮ
