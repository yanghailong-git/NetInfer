## 关键模块说明

### 1. Focus → 6×6 Conv (stride=2)
YOLOv5 v6.0+ 版本将原先的 `Focus`（切片+拼接+Conv）替换为 **6×6 卷积 stride=2**，效果等价但速度更快。`model.0` 就是这个操作。

### 2. C3 模块（CSP Bottleneck with 3 convolutions）
这是 YOLOv5 的核心单元：
- 两条分支：主干路经过 `n` 个 Bottleneck（残差块），旁路直接 1×1 卷积
- 最后 `cat` 拼接 + 1×1 融合
- `model.2/4/6/8/13/17/20/23` 都是 C3 模块

### 3. SPPF（Spatial Pyramid Pooling - Fast）
`model.9`：
- 替代了原版 SPP，用 **3 次 5×5 MaxPool 串联** 代替多尺度池化
- 输出与输入串联后通过 1×1 卷积融合

### 4. FPN + PAN（Neck）
- **FPN（自顶向下）**：`model.10→11→13` 和 `model.14→15→17`，通过 Upsample 融合深层语义特征到浅层
- **PAN（自底向上）**：`model.18→20` 和 `model.21→23`，通过 Conv(s=2) 下采样，将浅层位置特征回传深层

### 5. Detect 检测头
`models.yolo.Detect`（代码中对应 `YoloDetectLayer`）：
- 3 个分支分别对应 3 个检测尺度（80×80, 40×40, 20×20）
- 每个分支使用 **1×1 卷积** 将通道数映射到 `255 = 3 × (5 + 80)`
- 最终将 3 个尺度的预测展平拼接为 `(batch, 25200, 85)`
